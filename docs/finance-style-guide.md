# Finance Style Guide — 금융 표현 규칙

## 금지 표현 (자동 차단)

`forbid-financial-advice.sh` 훅이 다음 패턴을 차단한다. 일부러 우회 금지.

| 금지 | 이유 |
|---|---|
| "반드시 사세요", "지금 사야", "사야 합니다" | 매수 조언 |
| "절대 팔지 마", "팔면 안 됩니다" | 매도 조언 |
| "100% 수익", "확실한 수익", "보장된 수익" | 사기성 표현 |
| "원금 손실 없음", "리스크 없는" | 사실 왜곡 |
| "몰빵", "단타로 ~억", "한 종목만 사면" | 도박적 권유 |
| "지금 안 사면 후회", "마지막 기회" | FOMO 자극 |
| "내부 정보", "관계자에 따르면 (출처 무)" | 검증 불가 |

## 허용 표현 (사실 기술)

- "X 의 주가는 30일간 −22% 하락했다" — 사실
- "Y 분기 매출은 컨센서스 대비 8% 미달했다" — 사실
- "이는 통상적으로 부정적 신호로 해석된다" — 일반론, 단정 X
- "투자자는 자신의 리스크 허용 범위 내에서 판단해야 한다" — 권유 X

## 면책 문구 (모든 리포트 말미에 필수)

```markdown
---

## ⚠️ 면책

이 리포트는 교육 및 정보 제공 목적으로 제작되었습니다.
특정 종목의 매수·매도를 권유하지 않으며, 투자 자문이 아닙니다.
모든 투자 결정과 그 결과는 투자자 본인의 책임입니다.
리포트 내용은 작성 시점 (YYYY-MM-DD) 의 공개 정보를 바탕으로 하며,
이후 변경된 사실은 반영되지 않을 수 있습니다.
```



- 최종 HTML에서는 별도 큰 섹션 제목 없이 하단 파인프린트 형태로 렌더할 수 있다.
- 즉 draft에는 면책 문구를 유지하되, 사용자 화면에서는 최소한의 footer note처럼 보이게 처리해도 된다.

## 출처 표기 방식

- 본문에는 `[^1]`, `[1]` 같은 각주 번호를 넣지 않는다.
- 모든 사실은 `research/` 근거로 검증한 뒤, 리포트 마지막 `References` 섹션에만 출처를 모은다.
- References 형식:

```markdown
## References

- 매체명 — 기사 제목 (YYYY-MM-DD) — URL — Tier A|B|C
```

## 수급 데이터 의무 포함

- 한국 상장 종목 리포트는 **외국인, 기관, 개인 수급 추이**를 반드시 포함한다.
- 미국 등 해외 주식은 한국식 개인·외국인·기관 일별 수급 섹션을 만들지 않는다.
- 해외 주식에서 기관 보유자, 내부자 거래, 애널리스트 의견 같은 대용 지표를 사용할 수는 있지만, 사용자가 명시적으로 요청하지 않으면 별도 `수급과 투자자 구도` 섹션을 만들지 않는다.
- 표현 방식은 자유다.
  - 불릿 3줄
  - 표 1개
  - 차트 1개
- 한국 상장 종목에서 세 주체 중 하나라도 빠지면 불완전한 초안으로 본다.

## 용어 표기 통일

| 영문 | 한국어 (리포트 본문) | 차트 라벨 |
|---|---|---|
| Earnings | 실적 | 실적 |
| Revenue | 매출 | 매출 |
| EPS | 주당순이익 (EPS) | EPS |
| PER | 주가수익비율 (PER) | PER |
| Market Cap | 시가총액 | 시총 |
| Volume | 거래량 | 거래량 |
| Volatility | 변동성 | 변동성 |
| Bear / Bull | 약세 / 강세 | 약세 / 강세 |

영문 약어는 처음 등장 시 풀어 쓰고 괄호. 두 번째부터는 약어만.

## 숫자 표기

- 통화: `$123.45`, `₩123,456`, `¥1,234`
- 비율: `−22.4%` (소수 1자리 기본, 부호 명시)
- 큰 수: `1.2조원`, `8.4억달러` (한국어 리포트 기본 단위 우선)
- 거래량: `1.2M주`, `850K주`
- 시각/시간: `2026-05-12 (현지)` 또는 `2026-05-12 KST`

## 출처 신뢰도

`research/` 의 모든 출처는 등급을 표기한다.

- **A** — 1차 자료 (기업 공시, SEC 10-K/10-Q, 거래소, 중앙은행, IMF)
- **B** — 검증된 매체 (Bloomberg, Reuters, WSJ, FT, 한국경제, 매경, 연합뉴스)
- **C** — 일반 매체 / 블로그 / 분석가 의견
- **D** — 검증 어려움. 리포트에서 인용 금지.

본 리포트는 **A/B 출처에서 사실을 가져오고, C 는 의견이라 명시할 때만 사용**한다.
